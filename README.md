# SEO AEO GEO Academy

A static learning prototype for SEO, AEO, GEO, and AI Search Optimization.

## Tech Stack
- HTML, CSS, JavaScript, JSON
- No backend, database, auth, or AI APIs

## How to Run
1. Clone the repo
2. Open `index.html` in a browser
3. Or deploy on Netlify:
   - Connect GitHub repo
   - Build command: none
   - Publish directory: `.`

## Structure
- `index.html` – Homepage
- `lessons.html` – Lesson index
- `lesson.html` – Single lesson
- `glossary.html` – Glossary + FAQ
- `css/style.css` – Styles
- `js/app.js` – Main logic
- `js/utils.js` – Helpers
- `data/*.json` – Content
- `public/*` – SEO assets
